我安装的Obsidian插件

1. Advanced Tables

用于对表格进行增强，我可能用到的主要作用就是将表格中的元素居中或者靠左靠右。

2. Calendar

日历插件，可以点击日历上的时间创建当天的笔记，对我而言最大的作用可能就是个日历。

3. Tasks

一个用于创建任务的插件，需要先按住ctrl+p，然后在输入框中输入task，然后去使用，对我这种懒狗形式主义大于实际作用。

4. File Explorer Note Count

用于展示每个文件夹下有多少文件，挺实用的。

5. Recent Files

用于展示最近打开的文件，好用，爱用。

6. Better Word Count

当鼠标选中文字的时候，可以在右下角展示选中文字的个数，没啥用，单纯好玩。

7. Open In New Tab

一个和vscode一样恶心的设置，点击另外一个文件的时候会在一个窗口打开，有了这个就解决了，好用。

8. Paste URL into selection

就是在外面复制链接后，选中某段文字后，按住ctrl+v，就可以将那段文字变成超链接。

9. Homepage

可以设置一个页面作为obsidian启动后的首页，准备等会自己定制一个。

10. Custom Frames

挺牛逼的一个插件，可以将外面的网页通过obsidian打开，进入插件后，点击add frame，然后在新建出的frame中点击show settings，进行编辑页面，在编辑页面中，可以编辑需要添加的网址的名字，地址以及图标。

11. Style Settings

一个主题美化的插件，还是挺牛逼的，不出意外的话，很久不要改动了。

12. Manual Sorting

实现文件夹的上下拖动，牛逼。

13. File Hider

当想要隐藏一些文件夹的时候可以用到它。

14. Image Toolkit

当预览模式下，点击图片可以放大查看，牛逼。

15. Mousewheel Image zoom

通过alt+鼠标滚轮实现图片大小的控制，牛逼。

16. YouHaveBeenStaring

在右下角展示你已经打开obsidian多长时间，纯yy用。
